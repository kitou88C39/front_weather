#### 使用技術

React

TypeScript

Recharts

Google Maps API (MapsJavaScriptAPI, GeocodingAPI)

OpenWeather API

#### 機能一覧

Geolocation による、アプリ起動時の現在地の取得

Geocoding API での地名検索からの緯度経度取得

Maps JavaScript API でのマップにエリアの表示

OpenWeather API での、検索した地域の天気情報取得・表示
